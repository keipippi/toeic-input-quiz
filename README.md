# TOEIC Input Quiz

Streamlit製のTOEIC入力式単語練習アプリです。

英単語を見て日本語で答える練習、日本語を見て英単語で答える練習、カード暗記、10問連続演習、ユーザー別の履歴管理に対応しています。

## 主な機能

- 入力式クイズ
- 英→日 / 日→英 / ランダム出題
- 10問連続モード
- Quizlet風カードモード
- 苦手単語の優先出題
- ユーザー名ごとの履歴・成績分離
- 復習期限ベースの出題
- 単語追加フォーム
- words.csv の品質チェック
- 発音ボタン
- スマホ向けレイアウト

## 起動方法

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで表示されたURLを開くと利用できます。

## 使い方

1. サイドバーでユーザー名を入力します。
2. レベルを選びます。
   - 600だけ
   - 600+700
   - 全部
   - 手動で選ぶ
3. 出題方向を選びます。
   - 英→日
   - 日→英
   - ランダム
4. 出題モードを選びます。
5. 答えを入力して判定します。

## 出題モード

### 全単語

選択したレベルの単語から出題します。

### 10問連続

10問を1セットとして出題します。進捗と最後の結果を表示します。

### カード

単語カード形式で練習します。

- 英→日: 表が英単語、裏が意味
- 日→英: 表が意味、裏が英単語
- ランダム: カードごとに方向を切り替え

`覚えた` を押すと正解扱い、`苦手` を押すと不正解扱いで履歴に保存されます。

### 間違えた単語だけ

過去に不正解またはほぼ正解だった単語だけを出題します。

### 復習期限の単語

履歴に保存された `next_review` が今日以前の単語を出題します。

## 苦手単語の優先出題

`苦手単語を優先` をオンにすると、履歴をもとに出題されやすさを調整します。

優先度には以下を反映しています。

- 不正解回数
- ほぼ正解回数
- 正解率
- 直近の結果
- 復習期限の超過

## 単語データ

単語データは `words.csv` に保存します。

必要な列は以下です。

```csv
word,meaning,accepted_answers,example,note,level,pos,example_ja,ipa
```

| 列名 | 内容 |
| --- | --- |
| word | 英単語 |
| meaning | 主な日本語訳 |
| accepted_answers | 許容する別解 |
| example | 英語例文 |
| note | メモ |
| level | 600 / 700 / 800 |
| pos | 品詞 |
| example_ja | 例文の日本語訳 |
| ipa | 発音記号 |

`ipa` と `note` は空欄でも利用できます。

## 単語追加

アプリ内の `単語追加` タブから `words.csv` に単語を追加できます。

追加時には以下をチェックします。

- 必須項目が入力されているか
- 同じ英単語がすでに登録されていないか

## 品質チェック

`品質` タブで単語データの状態を確認できます。

- 登録語数
- 重複数
- 要確認件数
- 発音記号未入力数

## 履歴データ

ユーザーごとの履歴は以下の形式で保存されます。

```text
history_<ユーザー名>.csv
```

履歴には解答日時、単語、出題方向、結果、次回復習日などが保存されます。

## ファイル構成

```text
.
├── app.py
├── history.py
├── quiz.py
├── requirements.txt
├── README.md
├── words.py
└── words.csv
```

- `app.py`: Streamlitの画面
- `words.py`: 単語CSVの読み込み、追加、品質チェック
- `history.py`: ユーザー別履歴の読み込み、保存、復習日計算
- `quiz.py`: 解答判定、出題対象の絞り込み、苦手単語の優先度計算
